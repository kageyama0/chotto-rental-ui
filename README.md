# 構成
フロントエンド : Next.js + TypeScript + Mantine (https://github.com/kageyama0/chotto-rental-ui)
バックエンド : Go + Gin (https://github.com/kageyama0/chotto-rental)
DB : Postgresql

# セットアップ

## .envを作成
  
  ```
  cp .env.example .env
  ```

## .envを編集


## パッケージのインストール

  ```
  pnpm install
  ```

## 起動
  
    ```
    pnpm dev
    ```
